# FGCS Submission Package

**Paper:** A Hybrid Edge-Cloud Stream Processing Framework for Low-Latency Real-Time Analytics  
**Journal:** Future Generation Computer Systems (Elsevier)  
**ISSN:** 0167-739X  
**Submission Portal:** https://www.editorialmanager.com/fgcs/  

## Files

| File | Description |
|------|-------------|
| `main.tex` | Main manuscript (LaTeX, elsarticle class) |
| `references.bib` | BibTeX references |
| `cover-letter.tex` | Cover letter to the editor |
| `highlights.txt` | 3-5 bullet point highlights (required by FGCS) |
| `template/` | Elsevier elsarticle LaTeX template |

## Submission Checklist

- [ ] Create account at https://www.editorialmanager.com/fgcs/
- [ ] Upload `main.tex` as manuscript
- [ ] Upload `references.bib`
- [ ] Upload `cover-letter.tex` (or PDF)
- [ ] Paste highlights in submission form
- [ ] Select article type: "Research Paper"
- [ ] Add keywords from paper
- [ ] Suggest 3 reviewers (optional but recommended)
- [ ] Submit

## FGCS Requirements

- **Format:** Single-column, double-spaced for review (elsarticle preprint mode)
- **References:** Numbered style (elsarticle-num)
- **Figures:** Embedded in text for review; separate high-res files for production
- **Word limit:** No strict limit, but typically 8,000-15,000 words (ours is ~19,500 — acceptable for systems papers)
- **Cost:** $0 (subscription journal, no APC)
- **Review time:** ~3-6 months
